AccessToken (Optional)
Type: SystemString
The token obtained after the login flow in exchange for the `RequestToken` . Pre-login, this will default to None,but once you have obtained it, you should persist it in a database or session to pass to the Kite Connect class initialisation for subsequent requests.
Root (Optional)
Type: SystemString
API end point root. Unless you explicitly want to send API requests to a non-default endpoint, this can be ignored.
Debug (Optional)
Type: SystemBoolean
If set to True, will serialise and print requests and responses to stdout.
Timeout (Optional)
Type: SystemInt32
Time in milliseconds for which the API client will wait for a request to complete before it fails
Proxy (Optional)

To set proxy for http request. Should be an object of WebProxy.
Pool (Optional)
Type: SystemInt32
Number of connections to server. Client will reuse the connections if they are alive.
